# Harper PMO Framework

> Practical first.
>
> Standard Second.

## About

Harper PMO Framework는 제조업 개발조직을 위한 실전 PMO Framework를 연구하고 구축하는 프로젝트입니다. 

이 Repository는 Framework를 개발하고 발전시키는 과정(Lab)을 기록합니다. 

최종 Framework는 GitBook에서 출판합니다. 

## Vision

제조업 개발조직이 실제 프로젝트에서 바로 활용할 수 있는 실용적인 PMO Framework를 만든다. 

## Purpose

- PMP 학습
- PMBOK 연구
- 제조업 개발 프로젝트 실무 검증
- Harper PMO Framework 구축

## Architecture
| Platform | Purpose | Owner |
| ------ | ------ | ------ |
| GitHub | Framework Lab | Harper |
| GitBook | Published Framework | Harper |

## Framework 발전 사이클

1. PMBOK과 PMP를 학습한다.
2. 실제 프로젝트에서 검증한다.
3. Framework를 개선한다.
4. GitBook에 출판한다.
