## About

Harper PMO Framework는 제조업 개발조직을 위한 실전 PMO Framework를 연구하고 구축하는 프로젝트입니다. 

이 Repository는 Framework를 만드는 과정(Lab)을 기록합니다. 

최종 Framework는 GitBook에서 출판압니다. 

## Purpose

- PMP 학습
- PMBOK 연구
- 제조업 개발 프로젝트 실무 검증
- Harper PMO Framework 구축
