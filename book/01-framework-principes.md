# 01. Framework Principes

* Framework는 실제 프로젝트 경험을 기반으로 발전시킨다.
* PMBOK는 기준(Standard)으로 활용하며, 그대로 복사하지 않는다.
* 새로운 PMBOK 개념은 실무와 연결하여검증한 후 Framework에 반영한다.
* Framework의 변경은 이유와 근거를 함께 기록한다.
* Framework는 지속적으로 개선되는 Living Framework이다.
* Framework는 특정 회사의 운영 방식이 아니라, 제조업 개발조직에서 재사용 가능한 형태로 작성한다.
* Framework는 도구(Tool)가 아닌 원칙(Principle)을 우선한다.
